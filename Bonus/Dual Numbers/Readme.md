# Реализация класса для арифметики дуальных чисел

Простая библиотека для работы с **dual numbers** на C++.  
Реализует базовые арифметические операции (+, −, ×, ÷), а также сравнения и ввод/вывод.  
Может использоваться в задачах автоматического дифференцирования.

## Структура проекта

```
├── CMakeLists.txt
├── src/
│   ├── DualNumbers.h   # Заголовочный файл с реализацией класса
│   ├── DualNumbersMath.h # Заголовочный файл с реализацией функций
│   └── DualNumbers.cpp
└── tests/
    └── DualNumbersTests.cpp   # Тесты функцианала
```

В библиотеке `DualNumbers` реализованы базовые арифметические операции.  
Для работы с функциями типа экспоненты, логарифма, тригонометрии и корня создано отдельное пространство имён `dual`.

---

## Подключение

```cpp
#include "DualNumbers.h"
#include "DualNumbersMath.h"
using namespace dual;
```
Релизованные функции:
| Функция     |
| ----------- |
| `exp(x)`    |
| `log(x)`    |
| `sin(x)`    |
| `cos(x)`    |
| `sqrt(x)`   |
| `pow(x, y)` |
